# Instagram Clone

Instagram Clone build in **react-native using Expo CLI** and **firebase** with basic features like: 
- **Authentication**
- **Post Image using urls with Caption**
- **Like Posts**
- **Realtime Data**
- **Authentication**
- **Navigations**
